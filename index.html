Rock Hill Women's Art Initiative

This was a final team project at Winthrop. The site design and styling was done by my partner and I did all the JavaScript/interactivity, including an art map, project/event forms, backend PHP REST API, frontend AJAX to communicate with the REST API, and design and implementation of the MySQL database.

  • Client: Winthrop University
  • Date: Dec 2015
  • Technologies: HTML, CSS, jQuery, JavaScript, PHP, MySQL

Waiting to Cross - The Game

This was an interesting one. The idea was to create a site that was not meant to be just viewed from your desk, but instead had to be contextual.

For this, I created a game called "Waiting to Cross" and affixed it to an actual crosswalk light. There's also a makeshift button over the crosswalk button so that when you push it it starts the game and signals the crosswalk.

  • Client: Winthrop University
  • Date: Oct 2015
  • Technologies: HTML, CSS, jQuery, JavaScript
